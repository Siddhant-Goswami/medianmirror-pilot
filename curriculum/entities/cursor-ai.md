## Overview
Cursor is an AI-powered code editor designed for iterative codebase refinement. In the 100x ship cycle, it is the tool used after exporting a Lovable prototype to GitHub — it handles the refinement, debugging, and scaling work that Lovable cannot do.

## Key Contributions / Role
- AI-first code editor built on VS Code, with deep codebase context
- Step 5 in the 9-step ship cycle: receives Lovable output via GitHub, handles refinement
- Distinct role from Lovable: Lovable = greenfield generation; Cursor = iteration and scaling
- Enables developers to vibe-code refinements with engineering fundamentals as the quality filter
- Used alongside Claude Code for more complex agentic coding tasks
